# 🎯 Number Guessing Game

## Description
A beginner Python game where the player guesses a secret number.

## Features
- User enters guesses
- Program tells whether the guess is too high or too low
- Counts the number of attempts
- Ends when the correct number is guessed

## Concepts Used
- Variables
- Input/Output
- Conditionals (if, elif, else)
- While loop

## How to Run
```bash
python guessing_game.py
```
